## 🌱 About This Project

This repository hosts our **final project** for the **Programmation Masterlabs** at [TUMO Paris](https://paris.tumo.org/), led by Luka Piskorec.

**Our goal**: Build a **procedural 3D fantasy map generator** using JavaScript and noise algorithms. Inspired by games like Minecraft, we want to create unique, seed-based 3D maps with realistic terrain and fantasy aesthetics.

⚠️ **This project is currently under construction.** Code, features, and documentation are being actively developed.

---

## 🎨 Project Vision

- **Seed-based generation**: Enter a seed, get a unique map.
- **Noise-driven terrain**: Use [XXXXXXXXX] noise to create mountains, rivers, forests, etc.
- **3D rendering**: Visualize the map in 3D with elevation, textures, and lighting.
- **Fantasy aesthetic**: A blend of classic fantasy maps and modern procedural generation.
